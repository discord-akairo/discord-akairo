## Installation

Requires Node 12+ and Discord.js v12.  

*discord-akairo*  
`npm install greysilly7/discord-akairo` or `yarn add greysilly7/discord-akairo`

*discord.js*  
`npm install discord.js`

*sqlite (optional)*  
`npm install sqlite`

*sequelize (optional)*  
`npm install sequelize`
